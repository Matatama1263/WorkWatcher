# WorkWatcher

> [!important]
> 이 프로그램은 관리자권한을 필요로한 프로그램 강제종료 기능을 사용합니다.
> 본 프로젝트에 기여한 개발자는 사용자가 해당 프로그램을 사용하여 생긴 피해 *(예: 저장하지 못한 데이터의 손실)* 에 대하여 책임을 지지 않습니다.

WorkWatcher는 사용자가 능동적으로 작업시간을 관리하는데에 도움을 주는 프로그램으로 다른 프로그램들을 사용하는 시간을 측정하고 제한하며 통계를 생성합니다.

## 설치 방법

### 다운로드
[최신 릴리스 다운로드](https://github.com/Matatama1263/WorkWatcher/releases/latest)

### 설치 옵션

**옵션 1: 자체 포함형 (추천)**
- `WorkWatcher-SelfContained-win-x64.zip` 다운로드
- .NET Runtime 설치 불필요
- 압축 해제 후 바로 실행 가능

**옵션 2: Framework 의존형**
- `WorkWatcher-FrameworkDependent-win-x64.zip` 다운로드
- [.NET 8 Runtime](https://dotnet.microsoft.com/download/dotnet/8.0) 설치 필요
- 파일 크기가 작음

## 시스템 요구사항

- Windows 10 (버전 1809) 이상 또는 Windows 11
- x64 프로세서
- .NET 8 Runtime (Framework 의존형의 경우)

## 주요 기능 설명

### 프로그램 사용 시간 측정
사용자가 등록한 프로세스를 직접 사용하는 시간들을 측정하고 기록합니다.
이 프로그램들은 작업 프로그램과 딴짓 프로그램으로 분류하여 관리할 수 있습니다.
딴짓 프로그램에는 **강한 감시**를 설정할 수 있습니다. 강한 감시가 적용된 프로그램은 측정기능을 사용하는 동안 실행시킬 수 없게 됩니다.

### 할당량, 처벌기능
사용자는 **작업시간 할당량**과 **딴짓 허용시간**을 설정할 수 있습니다.
만약 사용자가 딴짓 프로그램을 사용한 시간이 딴짓 허용시간을 초과하면 처벌이 발동하여 등록해놓은 모은 딴짓 프로그램에 **강한 감시**가 적용됩니다.
반대로 사용자가 작업시간을 사용할 시간이 할당량을 초과하면 모든 딴짓 프로그램에 적용되어있던 모든 **강한 감시**가 해제됩니다.

### 통계
사용자가 이 프로그램을 사용한 기록과 데이터들을 저장하고 이를 활용한 통계를 생성하여 사용자에게 제공합니다.

## 사용 방법
- **설정**에서 작업을 할 때 사용하는 프로그램과 딴짓을 할 때 사용하는 프로그램들을 등록하고 원하는 작업시간 할당량과 딴짓을 허용할 시간을 입력합니다.
-  중앙의 **커다란 전원버튼**을 클릭하여 새로운 새션을 시작하거나 현재 세션을 종료할 수 있습니다.
-  **통계**에서 지금까지의 기록을 확인할 수 있습니다.
